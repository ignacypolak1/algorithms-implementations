class structures.graph.Node(name: str, neighbour: Node | None = None, cost: int | None = None)

Bases: object

A class for representing graph node

Nodes are fundamental parts of a graph structure (e.g., in graph theory or network routing). Each node has a name, can have multiple neighbours with associated costs, and can be designated as a root node.

Variables:

  • name – Unique name of the node

  • isRoot – True if the node has no parent, False otherwise. Automatically determined.

  • neighbours – List of tuples containing neighbours and the cost to reach them.

  • level – The level or depth of the node in the graph. Root nodes are at level 0.

Parameters:

  • name (str) – Name of the node

  • neighbour (Node) – Neighbour/Parent of the node

  • cost (int) – Cost of path between the node and given neighbour node
